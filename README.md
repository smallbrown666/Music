# 网易云音乐web app

  使用vue cli3构建的SPA移动端网页,有搜索、播放、和歌单功能 <br>
  api来源：网易云API  Node.js版 <br>

### 预览地址：

​	                   http://www.mxjx.xyz/music

## 技术栈：
  vue+vue-router+vuex+axios;部分UI用到vux

新增了better-scroll 插件 用于页面下拉弹窗

## 运行此项目：

  ```
 npm install
  ```
